Sobre o Projeto

Este é o website oficial do Porto Seguro Summer Geek, um projeto desenvolvido para ser a vitrine digital e o ponto de encontro da comunidade em torno dos nossos eventos. O site foi criado com o objetivo de oferecer uma experiência visual rica, dinâmica e de alta performance, capturando a energia e a diversão do universo geek.

🛠️ Stack Tecnológico e Animações
O projeto foi construído sobre uma base moderna para garantir velocidade e interatividade.

Categoria	Tecnologia	Uso Principal
Framework Principal	Next.js	Framework React que fornece SSR/SSG (Server-Side/Static Generation) para performance e routing.
UI	React	Biblioteca JavaScript fundamental para a construção da interface de usuário modular e reativa.
Animação	GSAP (GreenSock)	Biblioteca de animação de alto desempenho usada para criar movimentos de scroll, transições e interações fluidas e profissionais.
Animação	Lottie	Utilizada para renderizar animações vetoriais exportadas do After Effects (JSON), garantindo gráficos escaláveis e leveza no carregamento.
